# lpm - legendary package manager

## What is lpm?
Lpm, the legendary package manager, is the successor to apt for Debian and Debian-like packages.

## Installation
### HackerOS
hacker unpack lpm

### Ubuntu/Debian
wget -P /tmp https://raw.githubusercontent.com/HackerOS-Linux-System/lpm/main/install.sh && chmod +x /tmp/install.sh && /tmp/install.sh
